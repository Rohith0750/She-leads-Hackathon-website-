# She Leads Hackathon Website

Welcome to the official repository for the **She Leads Hackathon** website. This project is built to provide all necessary information about the hackathon, including tracks, timeline, sponsors, and more. It features a modern, responsive design with dark/light mode support.

## üöÄ Features

- **Modern UI/UX:** Clean and responsive design tailored for a hackathon event.
- **Dark/Light Mode:** Toggle between themes for comfortable viewing.
- **Key Sections:**
  - **Hero Section:** Introduction and call to action.
  - **About & Vision:** Information about the event's goals.
  - **Tracks:** Details on the hackathon themes and challenges.
  - **Timeline:** Schedule of events.
  - **Prizes:** Information on awards.
  - **Sponsors:** Showcase of our partners.
  - **Team:** Meet the organizers.
  - **College Map:** Location details.
  - **FAQ:** Frequently asked questions.

## üì¶ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rohith0750/She-leads-Hackathon-website-.git
   cd She-leads-Hackathon-website-
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Visit `http://localhost:5173` to view the application.
